Портреты
========

Реализация этапа "Портретные Башни" квеста ЛОЛ-2019. Живую версию можно увидеть [здесь](http://qbit.org.ua/lol/2019-quest/).

Видео, использованное в портретах, со оригинальным звуком: https://youtu.be/Y1RnfOtGGsY

Setup Environment
-----------------

```
$ npm install
```

Run
---

### Development Mode

```
$ npm run dev
```

### Static Export

```
$ npm run build
$ npm run export
```
